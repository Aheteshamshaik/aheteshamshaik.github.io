## 📌 Featured Projects

### 🔹 Tata Communications CPaaS Integration Connector
A full-stack middleware platform and web dashboard built from 0 to 1, integrating Tata Communications CPaaS services (DIGO & Kaleyra) with WooCommerce, Shopify, and HubSpot. Features a custom **Rules & Routing Engine** that maps inbound webhook events (`order.created`, `orders/fulfilled`, `contact.created`) to SMS/WhatsApp/Voice channels with dynamic merge-tag templating. Includes **real-time WebSocket broadcasting** for live activity feeds and delivery status monitoring, platform-specific plugins (WooCommerce in PHP, a serverless HMAC validator on AWS Lambda/Vercel for Shopify, and a custom HubSpot workflow action), and a **Sandbox Simulator** with mock payload delivery for credential-free testing.
**Stack:** Python · JavaScript · PHP · WebSockets · REST APIs · AWS Lambda · Vercel · HMAC Security

### 🔹 Smart Content Aggregator — AI-Powered NLP Web App
A full-stack AI web application built with Python Flask, SQLite, and vanilla HTML/CSS/JS. Implements **semantic search and personalized recommendations** using SentenceTransformers (all-MiniLM-L6-v2) vector embeddings and cosine similarity. Integrates HuggingFace Transformer pipelines for automated text summarization (DistilBART) and zero-shot article categorization (facebook/bart-large-mnli), with spaCy for NLP preprocessing. Includes 12+ REST API endpoints for ingestion, search, and live news fetching, backed by an automated web scraper (BeautifulSoup4) and RSS aggregator for real-time content.
**Stack:** Python · Flask · SQLite · SentenceTransformers · HuggingFace · spaCy · BeautifulSoup4

### 🔹 Pandora — Image Gallery Web App
A full-stack image hosting and gallery application built from scratch using Python Flask, SQLite, and vanilla HTML/CSS. Features secure file upload handling with filename sanitization and SQLite-backed metadata persistence, a live keyword search across title and description fields with dynamic filtering, and a responsive CSS Grid layout with hover animations.
**Stack:** Python · Flask · SQLite · HTML · CSS
